TruVue
TruVue is a very flexible high-performance platform for all kinds of panoramic images and interactive virtual tours. It allows easy creation, editing, and publishing of content. Virtual hot spots (such as picture or video ads, real time stock quotes, etc) can be added to interact with visitors during the tour. This product has been widely used in industries such as tourism (virtual sight seeing), real estate (real estate for sale) and government (digital city, software park, public facility guide) as well as used by online retailers, high end clubs, hotels, restaurants, exhibitions and shows.

Technology: Technology stack for this solution includes Spring, Hibernate, MySQL, Cassandra, jquery, iOS, Android, Rest Web Service, Ajax + Javascripts. V1.0 Beta release running live on Amazon Web Services (AWS) and Aliyun (for Chinese market).

Enterprise Cloud Drive
Enterprise Cloud Drive provides the functionalities of a cloud drive and includes end-to-end encryption on both client side and server (cloud side). All the files saved in the Enterprise Cloud Drive (both client side and server side) are encrypted. The encryption is transparent to the end user and the applications. As long as the user is authorized to access the file, the key for the client encryption will be sent down from the Enterprise Cloud Drive cloud and the encryption and decryption process are performed in the kernel mode.

Technology: The server architecture for the Enterprise Cloud Drive solution is based on HDFS/HBASE and OpenStack Storage Services. On the client side, it is developed using C/C++ in the Kernel mode on both Windows platform and Linux platform, and using Google protocol buffer + Boost asio for the network communication with the server.

A hosted SaaS model to drive consumers to your restaurant so you do not have to worry about any technology maintenance or support. Our platform is highly scalable and robust and can support tens of millions of users making reservations simultaneously in real time.

Technology: HTML5, Flash, Hibernate, MySQL, Cassandra, jquery+ Ajax, iOS, Android, Rest Web Service.
