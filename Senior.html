Senior Software Engineer, San Francisco, CA

    Responsibilities:
  • Build distributed systems using Java, Scala, Ruby on Rails, Redis, MongoDB, Lift, and AWS.
  • Participate in design and code reviews.
  • Rapidly fix bugs and solve problems.
  • Identify and address performance bottlenecks.
  • Integrate with other systems and tools.

    Qualifications:
  • Extensive experience programming Java or Scala.
  • Experience building software for the web end to end.
  • Disciplined approach to testing and quality assurance.
  • Passion for performance debugging and benchmarking.
  • 5+ years of industry experience.
  • BS or MS degree in Computer Science (or equivalent work experience).
